# NGUVU Agency — Agence de Communication Digitale

NGUVU est une agence de communication digitale premium basée au Cameroun, spécialisée dans la transformation numérique, l'intelligence artificielle agentique et la croissance stratégique.

## 🚀 Vision
Notre mission est de transformer l'énergie créative en performance mesurable, en alliant innovation technologique (IA) et authenticité culturelle africaine.

## 🛠 Technologies utilisées
Le projet est bâti sur une stack moderne et performante :
- **React 18** (avec TypeScript)
- **Vite** pour le build ultra-rapide
- **Tailwind CSS 4** (Design System & Utilitaires)
- **Framer Motion** (Animations fluides et immersives)
- **Lucide React** (Iconographie)

## ✨ Fonctionnalités Clés
- **Expérience Immersive** : Hero section avec effet machine à écrire et arrière-plan réseau IA dynamique.
- **Navigation Intelligente** : Menu adaptatif avec sidebar mobile élégante et barre d'infos dynamique.
- **Services IA** : Présentation des offres GEO (Generative Engine Optimization) et Agents Autonomes.
- **Composants Premium** : Carrousels de témoignages, modales de services détaillées et grilles de partenaires animées.
- **Performance UX/UI** : Design "Human-First" optimisé pour la conversion et l'esthétique luxe.

## 📦 Installation et Lancement

### Prérequis
- Node.js (version 18 ou supérieure recommandée)
- npm ou yarn

### Étapes
1. **Cloner le projet**
   ```bash
   git clone [url-du-repo]
   cd nguvu-agency
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```
   Le site sera accessible sur `http://localhost:3000` (ou port alternatif indiqué dans le terminal).

## 📁 Structure du Projet
- `/src` : Code source React
  - `App.tsx` : Composant principal (contenant la structure complète)
  - `index.css` : Configuration Tailwind et styles globaux
  - `main.tsx` : Point d'entrée de l'application
- `index.html` : Structure HTML de base
- `vite.config.ts` : Configuration de l'environnement Vite
